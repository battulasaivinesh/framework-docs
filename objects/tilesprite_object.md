# TileSprite Object

```
Library.Objects.TileSpriteObject({
  key: _image_key,
  alpha: _alpha,
  left: _left,
  top: _top,
  tileWidth: _width,
  tileHeight: _height,
})
```

## Arguments 

 + **key** - Object image key
 + **alpha** - Image alpha value 
 + **left** - left value of image
 + **top** - top value of image
 + **tileWidth** - width of tileSprite
 + **tileHeight** - height of tileSprite
